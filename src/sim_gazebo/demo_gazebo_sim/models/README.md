# Gazebo Fuel furniture models

These models were downloaded from the
[OpenRobotics collection on Gazebo Fuel](https://app.gazebosim.org/OpenRobotics/fuel/models)
for the 100 m² cleaning-robot test world. The original `model.config`
files are retained with each model.

| Local directory | Fuel model and version | License |
| --- | --- | --- |
| `bookshelf` | OpenRobotics / Bookshelf / 4 | CC0 1.0 |
| `chair` | OpenRobotics / Chair / 3 | CC BY 4.0 |
| `coffeetable` | OpenRobotics / CoffeeTable / 2 | CC BY 4.0 |
| `kitchencountertop` | OpenRobotics / KitchenCountertop / 3 | CC BY 4.0 |
| `normalbed` | OpenRobotics / NormalBed / 2 | CC BY 4.0 |
| `refrigerator` | OpenRobotics / Refrigerator / 1 | CC BY 4.0 |
| `sofa` | OpenRobotics / Sofa / 3 | CC BY 4.0 |
| `table` | OpenRobotics / Table / 4 | CC0 1.0 |
| `trashbin` | OpenRobotics / TrashBin / 2 | CC BY 4.0 |

Mesh URIs in the downloaded SDF files were changed from Fuel HTTP URLs to
relative paths so the simulation can run offline.
